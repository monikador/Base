Wyższa wydajność

CSS nowej generacji

Firefox Quantum zawiera nowy sposób renderowania CSS, napisany w języku Rust, który jest nowoczesny i oszałamiająco wydajny.

Więcej informacji

Nowe narzędzia

Nieaktywny kod CSS

Narzędzia dla programistów wyszarzają teraz deklaracje CSS, które nie mają wpływu na stronę. Po najechaniu na ikonę informacji wyświetlony zostanie przydatny komunikat o tym, dlaczego kod CSS nie jest stosowany, a także wskazówkę, jak naprawić problem.

Więcej informacji

Nowe narzędzia

Narzędzia dla programistów

Nowe narzędzia dla programistów oferują wysoką wydajność i możliwość personalizacji. Wśród nich jest najlepszy debuger języka JavaScript, napisany w React i Redux, działający z wieloma przeglądarkami.

Więcej informacji

Innowacyjne funkcje

Opanuj CSS Grid

Firefox to jedyna przeglądarka z narzędziami do tworzenia i projektowania za pomocą CSS Grid. Umożliwiają one wyświetlenie siatki i powiązanych z nią nazw obszarów, podgląd transformacji i wiele więcej.

Więcej informacji

Szybsze informacje

Panel czcionek

Nowy panel czcionek narzędzi dla programistów zapewnia szybki dostęp do wszystkich informacji o czcionkach używanych w elemencie. Zawiera także wartościowe informacje, takie jak źródło czcionki, waga, styl i nie tylko.
